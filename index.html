Portfolio

Take a look at some of my recent projects. You can find my github here.

Chef Genius

Rails, Backbone A clone of the popular site Rap Genius, my version allows the submission and annotation of cooking recipes, and commenting on those annotations.

  • Uses native JS to traverse and interact with the DOM on the node and element level.
  • Polymorphic associations and concerns on the Rails backend.
  • Uses event aggregator for communication between views.
  • Bootstraps initial data.

Cosmic Drift

Javascript, HTML5 Canvas Asteroids! Spaceships! Exciting interstellar action with movement reminiscent of the classic arcade game Asteroids.

  • Uses trig to calculate direction of movement and rotate ship.
  • Custom-shaped asteroids.
  • Advanced keybinding, with a cooldown for some actions, and continuous registration for others.

Node Chat

Node.js, socket.io Chatroom application built on a node backend that uses websockets for multi-client communication.

  • Supports custom usernames for users
  • Notifies users upon user entering or exiting chatroom.
  • Smart notification system ensures all users have up-to-date user list.

Tyrello

Backbone, Rails A clone of the website Trello. Users create boards, boards have lists, lists have cards. Cards and lists are both draggable and dropable.

  • Implements dragging and dropping using jQuery UI.
  • Uses HTML5 data attributes to persist changes to card/list order through drag and drop.
  • Uses jBuilder to customize data returned from API.

ActiveWreckered

Ruby A lite replication of Rails' ActiveRecord ORM features. Implements database persistence, mass assignment (soo Rails 3), querying, and associational methods.

  • Features heavy use of Ruby metaprogramming.
  • Uses several advanced raw SQL queries, with escaping to prevent injection attacks.

Tracks

Ruby Replicates Rails functionality, particularly the VC in Rails' MVC.

  • Supports Rails-style nested params, as well as query strings and route params.
  • Routes server request to the appropriate controller/action.
  • Controllers use binding to render ERB
  • Stores flash and session info in browser cookies.
